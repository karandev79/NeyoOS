# NeyoOS
A Personal Improvement and tracking system, to track your tasks, emotions, habits and more and allow you to analysis and improve yourself.

### Features
- Login system for protected access.
- Weekly grid planner to manage your week.
- Journal system, log your daily activity.
- Daily emotional and energy level tracking.
- Integrated task management with priority levels.
